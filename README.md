# Retail Demand Forecasting & Inventory Optimization

## 📊 Project Overview
This project predicts retail sales using machine learning and visualizes insights using Tableau.

## 🧠 Tools Used
- Python (Pandas, Scikit-learn)
- Tableau
- ChatGPT

## 🔍 Features
- Sales forecasting model
- Inventory risk analysis (Understock/Overstock)
- Interactive dashboard

## 📈 Dashboard Insights
- Sales vs Predicted Sales
- Store performance
- External factors impact

## 🚀 How to Run
1. Run Python script
2. Open Tableau dashboard
